# 🏅 Olympics Analysis Dashboard

## 📌 Project Overview
The **Olympics Analysis Dashboard** is an interactive Power BI project designed to analyze historical Olympic medal records and uncover trends across athletes, sports, countries, age groups, seasons, and gender participation. The project transforms raw Olympic data into meaningful insights through data cleaning, preparation, and visualization.

---

## 🎯 Objectives
- Analyze medal distribution among top athletes and countries.
- Identify the most successful sports and regions.
- Explore gender participation trends over time.
- Understand age-wise medal distribution.
- Compare medal counts across Summer and Winter Olympics.

---

## 🛠 Tools & Technologies
- **Excel** – Data cleaning, preprocessing, and handling missing values.
- **Power BI** – Dashboard creation and interactive visualizations.
- **Power Query** – Data transformation and shaping.

---

## 📊 Dashboard Features

### 🏆 Key Metrics
- **66 Sports Played**
- **230 Participating Countries**
- **271K Total Medals**

### 📈 Visualizations
- Top 10 athletes by medal count.
- Medal distribution by region.
- Medal count by sport.
- Gender-wise player participation trends over time.
- Medal distribution by gender.
- Age-wise medal count analysis.
- Region and season-wise medal comparison.

---

## 🔍 Key Insights
- The **USA** leads in total medal count among all regions.
- Athletics contributes the highest number of medals among sports.
- Male athletes account for approximately **72%** of total medals.
- Most medal-winning athletes belong to the **20–30 years** age group.
- Summer Olympics contribute significantly more medals than Winter Olympics.

---

## 📂 Project Workflow
1. Collected Olympic dataset.
2. Cleaned and transformed data using **Excel**.
3. Handled missing values and standardized columns.
4. Imported data into **Power BI**.
5. Created interactive visualizations and KPI cards.
6. Generated insights from historical Olympic records.
